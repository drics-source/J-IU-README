# 🚀 J-UI

**J-UI** is a convenient, all-in-one panel designed for seamlessly launching and managing **ComfyUI** and **SwarmUI** from Google Colab. 

---

## ✨ Features

*   **🖱️ Quick Launch:** Launch ComfyUI or SwarmUI with just a couple of clicks.
*   **📥 Integrated Downloader:** Easily download Models, LoRAs, VAEs, and more directly from Civitai, Hugging Face, and GitHub.
*   **☁️ Cloud Storage:** Native connection to Google Drive for saving and loading your assets.
*   **🚇 Web Tunneling:** Expose your workspace using Ngrok, Pinggy, or Gradio tunneling.
*   **⚙️ Profile Management:** Import and export your panel settings via a `.json` file for quick setup.
